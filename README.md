## 📂 Projetos 

### 🔄 [ETL — Dados Públicos](https://github.com/geraldomendz/etl-dados-publicos)
> **Python · Pandas · MySQL**

Pipeline de dados para ingestão, transformação e carga de dados públicos estruturados em JSON.

- Arquitetura **Medalhão** (Bronze, Silver, Gold) — separando dados brutos, tratados e consolidados
- Limpeza, padronização, validação de nulos e normalização de campos
- Persistência em banco relacional **MySQL** com foco em qualidade e escalabilidade

---

### 📊 [SQL — Fundamentos e Consultas](https://github.com/geraldomendz/sql-fundamentos-e-consultas)
> **MySQL**

Consultas analíticas para exploração e geração de métricas a partir de bases estruturadas.

- JOINs, GROUP BY, funções de agregação e filtros avançados
- Validação de integridade referencial e consistência dos dados
- Preparação de dados para suporte à tomada de decisão baseada em indicadores

---

### 📈 [Dashboard de Vendas — Power BI](https://github.com/geraldomendz/powerbi-dashboard-vendas)
> **Power BI · Power Query · DAX · Excel**
Dashboard interativo de vendas construído com foco em transformar dados brutos em indicadores para tomada de decisão.
- Modelagem no esquema **fato/dimensão** (fVendas, dClientes, dProdutos, dCalendario)
- Tratamento e padronização dos dados com **Power Query**
- Criação de métricas com **DAX**: Total Vendido, Lucro, Margem (%) e Impostos
